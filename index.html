Como funciona o multi-cliente
Cada cliente cadastrado tem seu próprio número de WhatsApp e prompt personalizado. O agente identifica de qual loja veio a mensagem pelo número conectado e usa a configuração correta automaticamente — sem intervenção manual.

Para conectar um segundo número: cadastre o cliente aqui e rode outra instância do servidor com o novo número escaneado no QR Code.

Subir o código no GitHub
Crie um repositório privado em github.com e execute no terminal:
cd seu-projeto

⚠️ Confirme que .gitignore inclui .env e auth_info_baileys/
2
Criar projeto no Railway
Acesse railway.app → Login with GitHub → New Project → Deploy from GitHub repo → selecione o repositório. O Railway detecta npm start automaticamente.
3
Configurar variáveis de ambiente
Railway → seu projeto → aba Variables → adicione cada uma:
GROQ_API_KEY         → sua chave do Groq (console.groq.com)
SUPABASE_URL         → URL do projeto Supabase
SUPABASE_SERVICE_KEY → chave service_role (não a anon!)
ADMIN_SECRET         → senha para este painel
PORT                 → 3000
4
Criar volume persistente (OBRIGATÓRIO)
Sem isso, toda vez que o Railway reiniciar o servidor, o WhatsApp vai desconectar e você precisará escanear o QR de novo.
Railway → seu projeto → New Volume → Mount path: /app/auth_info_baileys
5
Escanear o QR Code nos logs
Após o deploy (2-3 minutos), acesse: Railway → seu projeto → View Logs. O QR Code aparece ali. Escaneie com o WhatsApp do número que vai atender os clientes. Feito isso, não precisa mais repetir.
6
Conectar este painel ao Railway
Railway → Settings → Networking → Generate Domain. Você receberá uma URL como https://agents-xxxx.railway.app. Use essa URL no campo "URL da API" ao abrir este painel.

Arquivos extras para o painel funcionar 100%
1. Adicione ao seu index.js (após as outras rotas):
// CORS — permite que o painel HTML acesse de outra origem
app.use((req, res, next) => {
  res.header('Access-Control-Allow-Origin', '*')
  res.header('Access-Control-Allow-Headers', 'Content-Type, x-admin-secret')
  res.header('Access-Control-Allow-Methods', 'GET,POST,PATCH,DELETE,OPTIONS')
  if (req.method === 'OPTIONS') return res.sendStatus(200)
  next()
})

// Rota de conversas
import conversasRouter from './routes/conversas.js'
app.use('/admin/conversas', adminAuth, conversasRouter)

// Rota de simulação (para o painel testar o agente)
import { responderAgente } from './groq/agent.js'
import { getLoja } from './database/supabase.js'
app.post('/simulate', adminAuth, async (req, res) => {
  const { phone, text, loja_id } = req.body
  try {
    const loja = await getLoja(loja_id) // ou buscar por id
    if (!loja) return res.status(404).json({ error: 'Loja não encontrada' })
    const { resposta } = await responderAgente({
      loja, numeroCliente: phone,
      mensagemAtual: text, tipoMensagem: 'texto'
    })
    res.json({ reply: resposta })
  } catch(e) { res.status(500).json({ error: e.message }) }
})
2. Crie o arquivo src/routes/conversas.js:
import { Router } from 'express'
import { sb } from '../database/supabase.js'
const router = Router()

// Contatos únicos com última mensagem
router.get('/:lojaId/contatos', async (req, res) => {
  const { data } = await sb.from('conversas')
    .select('numero_cliente, nome_cliente, content, created_at')
    .eq('loja_id', req.params.lojaId)
    .order('created_at', { ascending: false })
  const map = new Map()
  ;(data || []).forEach(r => {
    if (!map.has(r.numero_cliente))
      map.set(r.numero_cliente, { numero: r.numero_cliente,
        nome: r.nome_cliente, ultima: r.content })
  })
  res.json(Array.from(map.values()))
})

// Histórico de um número
router.get('/:lojaId/:numero', async (req, res) => {
  const { data } = await sb.from('conversas')
    .select('role, content, tipo, created_at')
    .eq('loja_id', req.params.lojaId)
    .eq('numero_cliente', req.params.numero)
    .order('created_at', { ascending: true }).limit(80)
  res.json(data || [])
})

// Limpar histórico
router.delete('/:lojaId/:numero', async (req, res) => {
  await sb.from('conversas').delete()
    .eq('loja_id', req.params.lojaId)
    .eq('numero_cliente', req.params.numero)
  res.json({ ok: true })
})

export default router
